Create a Dyna Starter Theme Create a Dyna Child Theme Create Dyna Starter and Child

Dyna Starter Theme is for WordPress Theme Developer. Build your own WordPress Theme using PHP, JavaScript and SCSS or CSS.

Dyna Child Theme is for WordPress Website Owner. Change Style and Colors without coding or make small changes with PHP, JS and CSS.

Starter and Child is a ready-made set of both. The child theme is configured with dependency on your parent theme, fully white-labeled.
